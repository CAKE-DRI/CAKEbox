This page provides a template for creating a new Markdown contribution to CAKEbox.

## Getting started

1. Select the relevant theme.
2. Navigate to the appropriate folder.
3. Create a new `.md` file.
4. Add a clear, descriptive title.
5. Assign at least one tag:
   - Use an existing tag (see the list below when viewing this page in a web browser), or  
   - Create a new tag if needed.
6. Ensure your contribution aligns with a central theme.

## Types of resources

Contributions may take the form of:

- Step-by-step guides  
- Templates and checklists  
- Case studies  
- Example outputs  
- Reflections and lessons learned
